Principais formatações

Negrito / destaque

Nesse parágrafo temos uma frase em negrito usando a tag b (não semântica)

Nesta frase temos um termo em destaque usando a tag strong (semântica)

Nesta frase este termo está em itálico usando a tag I (não semântica)

Você também pode usar a tag EM (emphasis) para deixar um termo em itálico (semântico)

Podemos criar também um texto marcado usando a tag MARK

Você pode mudar a cor do texto marcado usando a tag mark e usando o parâmetro style dentro da tag mark

Texto grande e pequeno

Estamos criando um texto grande e outro pequeno nesse parágrafo.

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Texto subrescrito

Use a tag sup para fazer texto do tipo 52 + x = 30

Texto subscrito

Use a tag sub para fazer textos do tipo H20 é bom.
